# Amortable

> Amortization table calculator

## Usage

_Install package_

```bash
npm i -g amortable
```

_Run_

```bash
amortable --interactive
```

```bash
amortable --file FILE_PATH
```

## Run locally

### Requirements

NodeJS v10

### Commands

Specify file:

```bash
node index.js -f ./data.txt
```

Interactive mode:

```bash
node index.js -i
```
